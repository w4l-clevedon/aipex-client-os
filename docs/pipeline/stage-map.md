# Aipex Client OS Stage Map

## Stage -1 — Idea Qualification

**Agent:** Strategic Advisor  
**Purpose:** Decide whether the idea should be pursued as a plugin at all.

Checks:

- Plugin vs SaaS vs add-on vs theme extension vs existing solution
- Strategic fit with Aipex ecosystem
- Reuse of existing work
- High-level effort vs potential reward
- Whether the project deserves Stage 0

Gate:

- If plugin is not the best route, recommend the better route and stop.

Artifact: `00-idea-qualification.md`
