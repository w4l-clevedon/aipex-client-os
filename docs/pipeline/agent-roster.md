# Aipex Client OS Agent Roster

The pipeline uses specialist agents with clear responsibilities and handoffs. One agent works at a time unless explicitly directed otherwise.

## 1. Pipeline Director
Owns the whole project, maintains source of truth, controls stage progression, prevents context drift and confirms next action.

## 2. Strategic Advisor
Runs Stage -1. Decides whether the idea should be a plugin, SaaS, add-on, theme extension, workflow, or existing-tool implementation.

## 3. Project Intake Specialist
Runs Stage 0. Captures the brief, constraints, dependencies, buyer, must-haves, competitors and data sensitivity.

## 4. Market & Viability Analyst
Runs Stage 1. Performs demand, competitor, pricing, market, support and ROI analysis. Delivers GO, NO-GO or RESHAPE.

## 5. Product Manager
Runs Stage 2. Converts the idea into MVP, MoSCoW priorities, paid add-ons, packaging and first-customer scope.

## 6. WordPress Plugin Architect
Runs Stage 3. Designs the system architecture, WordPress/WooCommerce integration, file layout, hook map, security model and extensibility contract.

## 7. Senior Technical Writer
Runs Stage 3b. Converts architecture into developer-ready technical specification with classes, functions, settings, APIs, database and build order.

## 8. UX/Product Designer
Runs Stage 4. Designs admin and user flows, screen list, menus, onboarding, empty states, loading states and accessibility considerations.

## 9. Database Specialist
Runs Stage 5. Reviews schema, indexing, migrations, versioning, rollback, scalability and data integrity.

## 10. Senior WordPress Developer
Runs Stage 6. Builds to specification, follows WPCS, implements security, i18n, dependency checks, hooks and testable increments.

## 11. QA Engineer
Runs Stage 7. Creates and executes the test plan across unit, integration, browser, mobile, WooCommerce, PHP versions, multisite and plugin conflicts.

## 12. Security Engineer
Runs Stage 8. Performs focused security review for SQL injection, XSS, CSRF, privilege escalation, REST/AJAX abuse, file upload issues, encryption and secrets.

## 13. Code Reviewer
Runs Stage 9. Audits code quality and maintainability only. Does not rewrite. Returns findings ranked Critical, High, Medium and Low.

## 14. Marketing Strategist
Runs Stage 10. Defines ICP, USP, positioning, landing page outline, demo video outline, launch strategy, affiliate plan, SEO keywords and competitor comparison.

## 15. Pricing Strategist
Runs Stage 11. Defines tiers, licence model, renewal strategy, add-on pricing, sales channel, billing stack and revenue forecast.

## 16. Legal & Licensing Groundwork
Runs Stage 12. Maps GPL, third-party licences, distribution terms, draft EULA outline, privacy/GDPR touchpoints and solicitor flags.

## 17. Project Director / Handoff
Runs Stage 13. Produces roadmap, release plan, logs, handoff pack, next actions and project status.
